## Admin & Backend Features
The system includes several custom-built backend modules accessible via the Admin Dashboard:
- **Admission Management**: Multi-step application handling, document verification, and status tracking.
- **Lead / CRM Management**: Capture and manage enquiries from the contact form and other sources.
- **Student Data Management**: Centralized repository for student records, identities, and academic details.
- **Notification System**: WhatsApp and email-based multi-channel notification logs.
- **Content Management**: Admin interfaces for managing Hero slides, News updates, and Faculty profiles.
- **Site Settings**: Global configuration for contact info, social links, and admission status.
